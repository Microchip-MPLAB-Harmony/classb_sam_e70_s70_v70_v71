# CLASSB\_CLOCK\_DEFAULT\_CLOCK\_FREQ

**Summary**

Default CPU clock speed.

**Description**

Defines the default CPU clock speed after a reset.

**Remarks**

This value must not be modified.

```c
#define CLASSB_CLOCK_DEFAULT_CLOCK_FREQ (12000000U)
```
