# CLASSB\_INVALID\_TEST\_ID

**Summary**

Invalid test ID.

**Description**

Defines a constant to initialize the variable which holds the ID of the ongoing self-test.

**Remarks**

This value must not be modified.

```c
#define CLASSB_INVALID_TEST_ID (0xFFU)
```
