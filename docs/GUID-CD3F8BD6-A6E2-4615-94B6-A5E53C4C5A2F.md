# CLASSB\_FLASH\_CRC32\_POLYNOMIAL

**Summary**

CRC-32 polynomial.

**Description**

Defines the CRC-32 polynomial used for Flash self-test.

**Remarks**

This value must not be modified.

```c
#define CLASSB_FLASH_CRC32_POLYNOMIAL (0xedb88320U)
```
