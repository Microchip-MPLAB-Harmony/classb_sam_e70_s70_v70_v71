# CLASSB\_RESULT\_ADDR

**Summary**

Address of test results.

**Description**

This constant defines the address in SRAM where the test results are stored.

**Remarks**

This value must not be modified.

```c
#define CLASSB_RESULT_ADDR (0x20400000U)
```
