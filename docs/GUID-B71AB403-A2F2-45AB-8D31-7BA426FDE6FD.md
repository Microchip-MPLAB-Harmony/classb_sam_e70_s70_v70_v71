# CLASSB\_CLOCK\_TEST\_RTC\_CYCLES

**Summary**

Duration of the CPU clock test.

**Description**

Defines the duration of the CPU clock test in terms of RTC cycles.

**Remarks**

This value must not be modified.

```c
#define CLASSB_CLOCK_TEST_RTC_CYCLES (200U)
```
