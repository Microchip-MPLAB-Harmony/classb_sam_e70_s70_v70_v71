# CLASSB\_TEST\_STATE

**Summary**

Identifies Class B library test state.

**Description**

This data type is used to update flags which indicates whether a test has started or not.

**Remarks**

None.

```c
typedef enum
{
CLASSB_TEST_NOT_STARTED = 0,
CLASSB_TEST_STARTED = 1
} CLASSB_TEST_STATE;
```
